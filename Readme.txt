 |      Проект написано на php версії PHP 7.1.7. Тому прохання використовувати підходящу версію php.|
 |      Дя запуску проекта перейдіть в файл "config/db_params.php" , та налаштуйте підключення до   |
 |      своєї бази даних. Для цього ви повинні:                                                     |
 |      1) В змінній $DB_DSN і $DB_DSNF замість "127.0.0.1" написати host рідключення до бд.        |
 |      2) В змінній $DB_USER написати користувача підключення до бд.                               |
 |      3) В змінній $DB_PASSWORD написати пароль користувача.                                      |
 |      4) В змінній $DB_NAME написати імя бази даних яка буде використовуватися, якщо бази даних   |
 |          немає вона буде створена.                                                               |
 |                                                                                                  |
 |          Запустити локальний сервер:                                                             |
 |      1) зайти в папку з проектом і в терміналі прописати "php -S localhost:8000" (8000 - може    |
 |          бути  будь-який не зайнятий порт) також проект можна запустити на MAMP і тд.            |
 |      2) Ввести в браузері видану силку (під час першого запуску автоматично буде створена бд     |
 |          і таблиця яка потрібна для роботи).                                                     |
